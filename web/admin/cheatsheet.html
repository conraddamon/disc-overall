HARV 3000 Cheat Sheet
I need to get set up to record scores
  1. Go to the admin page for your tournament.
  2. Right-click the link "Enter result" and open it in another tab or window.
I need to record scores for an individual event (golf, accuracy, distance, SCF, or discathon)
  1. Go to the results entry page.
  2. Click on the event to select it.
  3. Select the division.
  4. Select the round. You are now ready to enter scores.
  5. Start typing a name into the Player box. Use the arrow keys to select a name, or keep typing until there is only one. When the name you want is highlighted, hit the Tab or Enter key.
  6. Type a score. Hit Enter.
  7. Repeat steps 5 and 6 for each score.
I need to record scores for a team event (DDC or freestyle)
  1. A DDC or freestyle score is what place they got in whatever group or pool they played in.
  2. Go to the results entry page.
  3. Click on the event (DDC or freestyle) to select it.
  4. Select the division.
  5. Select the round. You are now ready to enter scores.
  6. Start typing a team into the Team box. Use the arrow keys to select a team, or keep typing until there is only one. When the team you want is highlighted, hit the Tab or Enter key.
  7. Type a place. Hit Enter.
  8. Repeat steps 6 and 7 for each place.
I made a mistake
  1. Double-click a player or team name to delete a result.
The player whose score I was trying to enter does not appear in the list that pops up
  1. You need to register them. Go to the tournament admin page.
  2. Select the division.
  3. Start typing a name into the Name box. Use the arrow keys to select a name, or keep typing until there is only one. When the name you want is highlighted, hit the Tab or Enter key. If there is no match, type the full name.
  4. Click the Add button.
  5. Go to the results entry page.
  6. Click "refresh" so that the page will know about the new player.
I need to match up a DDC or freestyle team
  1. Go to the teams page for the tournament.
  2. Click on the event (DDC or freestyle).
  3. Select the division.
  4. If you want to be able to team players from different divisions, click "show all players".
  5. Drag an unteamed player onto another unteamed player to form a team.
  6. To make a freestyle threesome, drag a player onto a team.
